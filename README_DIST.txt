# MOVIEFY - Movie Recommendation Engine

Welcome to MOVIEFY! Follow these simple steps to get started:

## 1. Extract the ZIP
Make sure you have extracted all files from the ZIP folder into a regular folder on your computer.

## 2. Run Setup
Double-click the file named:
setup_wizard.bat

This will create a safe environment and install the necessary libraries for the AI to work. You only need to do this once.

## 3. Launch App
Double-click the file named:
START_MOVIEFY.bat

Your browser will automatically open, and you can start uploading your resume for movie matches!

---
System Requirements:
- Windows OS
- Python 3.8 or higher installed
- Internet connection (for initial setup and movie data)
